# hufbot
ATC Bot for hufworldwide.com (Shopify based website)

### Installation

hufbot requires [Node.js](http://nodejs.org/).

Setup:

```sh
$ cd hufbot
$ npm install
```

Configure information inside the `config.example.json` be sure to rename it to `config.json` when you're done
